# Projeto-interdisciplinar-
Nome do Projeto: Sistema para gestão e controle de doação de sangue para a rede de saúde municipal / Fatec Lins - Professor Antonio Seabra / 1º ADS / Wellyda Gonzales 
Descrição: O sistema é direcionado à rede municipal de saúde e tem como objetivo conectar, de forma rápida e eficiente, pessoas interessadas em doar sangue a pacientes que necessitam de transfusão. A plataforma identifica tipos sanguíneos compatíveis, facilita a comunicação entre doadores e solicitantes e contribui para agilizar o atendimento em situações de urgência, fortalecendo a rede de solidariedade e salvando vidas.
Técnologias Utilizadas: Figma, Paint, Visual Studio Code, HTML e CSS.
